# Android-WhiteBoard
Just a normal whiteboard app made with Kotlin

Added features:
  - Request permissions for read and write on external storage
  - Undo paths
  - Save drawings
  - Add custom pictures to background
  - Custom brush size
  - Hard-Coded colours
